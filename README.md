# Ponjeyakumar R - Portfolio

A modern, interactive portfolio website showcasing projects, skills, and experience with a unique terminal-inspired design.

## Features

- **Interactive Terminal**: Functional command-line interface with commands like `help`, `about`, `skills`, `projects`, `contact`, and more
- **Custom Cursor**: Animated cursor with ring effect
- **Theme Toggle**: Light and dark mode support
- **Responsive Design**: Works on mobile, tablet, and desktop
- **Project Carousel**: Smooth scrolling project showcase with pagination
- **Animations**: Scroll reveals, hover effects, and smooth transitions

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript
- No frameworks or build tools required

## Projects

1. **Lunex Terminal** - Command-line shell in C with built-in commands, pipes, redirections, and web-based terminal access
2. **Financial Risk Analytics Dashboard** - Analytics pipeline for VaR, volatility, and drawdown with Power BI
3. **ESG Portfolio Scoring Engine** - Data analytics web app that evaluates and ranks companies based on Environmental, Social, and Governance (ESG) metrics with interactive Plotly visualizations, deployed on Streamlit
4. **Secure URL Shortener** - RESTful backend with JWT auth, Redis caching, deployed on AWS
5. **Smart Energy Management** - IoT sensor monitoring system with Node.js backend
6. **Cloud Attendance System** - AWS serverless app with Lambda and SNS notifications
7. **Collaborative Doc Editor** - Real-time WebSocket editor with operational transform

## Getting Started

Simply open `index.html` in any modern web browser. No build process or server required.

For local development:
```bash
# Using Python
python -m http.server 8000

# Using npx
npx serve
```

Then visit `http://localhost:8000`

## Terminal Commands

- `help` - Show available commands
- `about` - Learn more about me
- `skills` - View technical skills
- `projects` - List all projects
- `project <n>` - Detailed info for project (1-7)
- `contact` - Get contact information
- `experience` - View work history
- `resume` - Download resume
- `source` - View source code
